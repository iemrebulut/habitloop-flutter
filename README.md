## Hakkında

HabitLoop, günlük alışkanlıklarını takip etmen için tasarlanmış sade bir mobil uygulamadır. Backend yoktur; tüm veri bellek içinde tutulur. Amaç; temiz mimari, Material 3 tasarım ve düşünülmüş state yönetimini tek bir portföy projesinde bir araya getirmektir.

- Günlük ilerleme kartı ve anlık tamamlama yüzdesi
- Kategoriye göre filtrelenebilen alışkanlık listesi
- Son 7 günün mini grafiği ile detay ekranı
- En uzun seri, haftalık başarı ve alışkanlık başına ilerleme barlarını gösteren istatistik ekranı
- Kategori, sıklık, hatırlatma saati ve renk seçimi içeren yeni alışkanlık formu
- Sistem ayarını izleyen aydınlık / karanlık tema
- Tam Türkçe arayüz

## Kurulum

```bash
git clone https://github.com/iemrebulut/habitloop-flutter.git
cd habitloop-flutter
flutter pub get
flutter run
```

**Gereksinimler:** Flutter 3.41+, Dart 3.11+, iOS 13+ / Android 7+.
